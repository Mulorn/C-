Программа была написана в Visual Studio и может не работать при отсутсвии Visual C++ 
При возникновении ошибки "Не удается продолжить выполнение кода, поскольку система не обнаружила MSVCP140D.dll..." 
скачать Visual C++ : https://www.microsoft.com/ru-ru/download/details.aspx?id=48145
Даже при условии, что у Вас 64 разрядная система, может потребоваться установить 32 битную версию.
